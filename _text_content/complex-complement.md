## [EXPOSITION]
**Complex complementary colors**

## [EXPOSITION]
This exercise shows the formula behind deriving complementary colors.

## [EXPOSITION]
1. Select the first shape and use the colormixer to set it to **140.55.205**

## [EXPOSITION]
2. Select the second shape and set its values so they are complementary (both values add up to 255.255.255 = **White**):

**Red** = **115** (255 − 140)

**Green** = **178** (255 − 55)

**Blue** = **50** (255 − 205)

## [EXPOSITION]
**Options:** You can do this for any color.

Mix the first color on the first shape (S1), then select the second shape (S2) and set the RGB values like this:

## [EXPOSITION]
S2 **Red** value = 255 − S1 **Red** value

S2 **Green** value = 255 − S1 **Green** value

S2 **Blue** value = 255 − S1 **Blue** value
